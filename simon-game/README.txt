A Simon game where you copy whatever buttons the computer randomly plays. Each button has a unique sound and color. The computer starts by playing one 
button and everytime you play the correct button, the computer adds another random button to the series. You have the option of playing the game using 
Strict Mode, which resets the sequence of buttons completely if you press a wrong button, and you start from the beginning. This game goes upto 20 steps, 
and after getting all 20 steps correctly, you win!

HTML5 | CSS3 | JavaScript | jQuery | Bootstrap | SASS
